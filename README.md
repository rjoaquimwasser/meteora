# Meteora
Projeto responsivo e mobile first com bootstrap desenvolvido durante o curso "Bootstrap5: crie uma landing page responsiva" da Alura

## Descrição 
O projeto é um e-commerce de uma loja de roupas.
Foi construído com as tecnologias HTML, CSS e Bootstrap5.
